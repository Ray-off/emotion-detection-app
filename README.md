# emotion-detection-app
Multi-class emotion detection using domain-specific BERT + Streamlit
## 🚀 Overview

This project is a **multi-class emotion detection system** built using a **domain-specific pretrained BERT model**. It classifies text into different emotional categories with high accuracy.

## 🔥 Features

* Multi-class emotion classification
* Domain-specific pretrained transformer (BERT)
* Streamlit web interface
* Probability visualization

## 📊 Model Performance

* Best Model: **Domain BERT**
* Accuracy: **96%**
* F1 Score: **0.96**

## 🧠 Models Compared

* Logistic Regression
* SVM
* Random Forest
* LSTM / GRU
* BERT
* Domain-specific BERT (Best)

## 🖥️ Run Locally

```bash
git clone https://github.com/YOUR_USERNAME/emotion-detection-app.git
cd emotion-detection-app
pip install -r requirements.txt
streamlit run app.py
```
